Project Brief

The ThermoSpark Protocol

01

Vapor Detection

MQ-2 sensor continuously samples the air every 10ms for electrolyte venting — the earliest chemical signature of thermal runaway.

02

Thermal Analysis

NTC thermistor data is compared against danger thresholds. A rapid rise gradient (dT/dt) triggers the pre-warning state before hitting critical.

03

MOSFET Isolation

The IRLZ44N MOSFET acts as a high-speed isolation switch. Digital pin goes LOW → gate closes → power supply to the load is severed in <1ms.

04

Kill Switch Active

Dashboard reflects POWER KILLED state with emergency red indicator. System remains isolated until manual reset — ensuring absolute safety redundancy.
